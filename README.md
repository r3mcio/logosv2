# Bydło-Trans Szczecinek

Professional website for Bydło-Trans Szczecinek, a cattle procurement company. Built with Vue 3, Vite, and Tailwind CSS.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
